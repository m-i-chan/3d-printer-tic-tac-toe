# 3D-Printer Tic-Tac-Toe Project

This project was developed for Framingham State's 2026 Science on State Street event. The goal is to develop a computer interface that will enable users to play tic-tac-toe using an Ender Creality 3 to display and update the game board.

Requires Python 3.12.
